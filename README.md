easyform
========

一款轻量级的jQuery表单验证插件
##Feature
* 基于jQuery，兼容各种浏览器
* 使用简单，体积小
* 通过修改配置可以实现多种功能    

##Usage

####首先你得在页面引用jQuery和jquery.easyform.js    
    <script src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
    <script src="js/jquery.easyform.js"></script>
####最基本的用法
    $('form').easyform();
这样在表单submit的时候easyform会校验form内所有的输入框以及下拉列表是否为空
